Die Leitidee des Kurses

„Dein Agent handelt — du behältst die Kontrolle" gilt hier unverändert. Neu ist nur deine Rolle: Du steuerst nicht mehr nur, du baust. Jede Lektion endet mit einem funktionierenden Zwischenstand, und kein Auftrag deines Agenten wird echt, bevor du ihn freigibst.

Dein Kursweg

  1. Verbinden → — n8n trifft den MCP-Zugang: erste Verbindung ohne KI, mit deinem Token.
  2. Dein Agent sucht → — Chat-Model anschließen, Tools begrenzen, den Marktplatz durchsuchen lassen.
  3. Dein Agent beauftragt → — ein echter Übungsauftrag entsteht und landet in deiner Freigabe.
  4. Verfolgen & Grenzen → — Status lesen, Grenzen verstehen, Fehler deuten.
