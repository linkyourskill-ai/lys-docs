Mission · Dein Agent handelt

Lass deinen Agenten suchen — du liest jeden Schritt mit

In Lektion 1 stand die Leitung. Jetzt kommt das Gehirn dazu: Du gibst deinem Agenten einen Auftrag in normaler Sprache, und er greift auf den Marktplatz zu. Aber vor jedem Tool-Aufruf fragt Claude Code dich zuerst um Erlaubnis — du siehst genau, was er vorhat, und du entscheidest.

Lektion 02 · ca. 7 Minuten · Wissen & Übung

Das kannst du danach

  • Deinem Agenten im Chat einen Suchauftrag geben — und im Erlaubnis-Dialog sehen, was er vorhat.
  • Im Protokoll nachlesen, welches Tool er mit welchen Werten aufgerufen hat.

Vom Zugang zum Agenten

In Lektion 1 hast du den MCP-Zugang mit Claude Code verbunden. Jetzt ist Claude Code dein Agent: Du schreibst ihm in normaler Sprache, was du brauchst, und er entscheidet, welche seiner Tools er dafür einsetzt. Starte Claude Code mit claude und gib ein:

Finde eine Reinigungskraft in Münster. Zeig mir die besten 3.

Der erste Erlaubnis-Dialog

Bevor dein Agent das Tool search_skillanbieter wirklich aufruft, hält Claude Code an und fragt dich:

Tool use lys - search_skillanbieter(query: "Reinigung Münster") (MCP) Do you want to proceed? ❯ 1. Yes 2. Yes, and don't ask again for lys - search_skillanbieter commands in dein-projekt 3. No

Das ist der Kern dieses Kurses. Der Dialog zeigt dir das Tool (lys - search_skillanbieter) und die Werte, die dein Agent senden will (hier die Suchanfrage). Nichts passiert, bevor du zustimmst. Für die Suche wählst du 1. Yes — es wird ja nur gelesen.

Achtung: der Auto-Modus schaltet deine Kontrolle ab

Steht in der Statuszeile ⏵⏵ auto mode on, ruft Claude Code Tools ohne zu fragen auf — der Dialog fällt weg, und damit deine wichtigste Kontrolle. Drücke Shift+Tab, bis der Hinweis verschwindet und du wieder im Standard-Modus bist, der bei jedem Schritt fragt. In diesem Kurs bleibst du immer im Standard-Modus.

Lies mit, was dein Agent tat

Nach deinem Yes arbeitet der Agent und fasst zusammen. Für eine ehrliche Rangfolge ruft er oft noch get_skillanbieter_ratings auf (auch dafür fragt er erst). Claude Code klappt solche Aufrufe zusammen:

Called lys (ctrl+o to expand)

Drücke Ctrl+O, um den Aufruf aufzuklappen — dort steht Wort für Wort, welches Tool mit welchen Werten lief und was zurückkam. Genau dieses Mitlesen ist deine Kontrolle in der Praxis: Du glaubst deinem Agenten nicht, du prüfst ihn.

Was zurückkam

Auf dem Test-Marktplatz sitzt nur Anna Behrens wirklich in Münster (0 km); die übrigen Anbieter sind 100 km+ entfernt. Weil noch niemand Bewertungen hat („Rating: N/A"), sortiert der Agent ehrlich nach Nähe statt nach Sternen — und sagt das auch dazu. Ein guter Agent erfindet keine Sterne.

Jetzt du

Prüf dich selbst — aus dem Gedächtnis abrufen baut echtes Behalten auf.

Wissens-Check

Was bedeutet ⏵⏵ auto mode on in der Statuszeile?

Im Auto-Modus überspringt Claude Code den Erlaubnis-Dialog und ruft Tools ohne deine Zustimmung auf — deine wichtigste Kontrolle fällt weg. Mit Shift+Tab kommst du zurück in den Standard-Modus, der bei jedem Schritt fragt. Schneller oder doppelt nachfragen tut der Modus nicht.

Primärquelle

Die Seite MCP Integration listet die Tools deines Agenten mit ihren Werten — dieselben, die im Erlaubnis-Dialog und beim Aufklappen mit Ctrl+O auftauchen.

Unsicher beim Mitlesen?
