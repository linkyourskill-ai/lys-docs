Dein Agent in Claude Code — du baust ihn selbst

Im Agentowner-Kurs hast du gelernt, KI-Agenten zu steuern. In diesem Kurs baust du deinen eigenen — im Terminal: Mit einem Befehl verbindest du Claude Code mit dem MCP-Zugang von LinkYourSkill. Am Ende chattest du mit einem Agenten, der für dich den Marktplatz durchsucht und Aufträge vorbereitet — und der dich vor jedem Schritt fragt. Freigeben musst immer noch du.

4 Lektionen · Deutsch · jeweils ca. 6–8 Minuten

Das nimmst du mit

  • Einen funktionierenden KI-Agenten in Claude Code — verbunden mit deinem Schlüssel, begrenzt auf die Tools, die du erlaubst.
  • Die eingebaute Kontrolle: Claude Code fragt vor jedem Tool-Aufruf um Erlaubnis — du liest mit und entscheidest.

Die Leitidee des Kurses

„Dein Agent handelt — du behältst die Kontrolle" gilt hier unverändert. Neu ist nur dein Werkzeug: Du steuerst nicht mehr nur, du baust — im Terminal. Und Claude Code fragt dich vor jedem Schritt um Erlaubnis. Kein Auftrag deines Agenten wird echt, bevor du ihn freigibst.

Zwei Wege, dasselbe Ziel

Dieser Kurs ist die Terminal-Variante von „Dein Agent in n8n". Beide bauen denselben Agenten am selben Marktplatz — wähle das Werkzeug, in dem du dich wohlfühlst: n8n visuell, Claude Code im Terminal. n8n brauchst du für diesen Kurs nicht.

Dein Kursweg

  1. Verbinden → — Claude Code trifft den MCP-Zugang: ein Befehl, dein Schlüssel, der Tool-Katalog. Noch ohne Chat.
  2. Dein Agent sucht — chatten, den ersten Erlaubnis-Dialog erleben, im Log lesen, was dein Agent wirklich tat. (folgt)
  3. Dein Agent beauftragt — ein echter Übungsauftrag entsteht: erst ablehnen, dann bewusst freigeben und im Messenger verwerfen. (folgt)
  4. Verfolgen & Grenzen — Status lesen, Tools dauerhaft begrenzen, Grenzen und Fehler verstehen. (folgt)
